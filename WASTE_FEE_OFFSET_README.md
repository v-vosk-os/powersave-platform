# Waste Fee Offset - ΟΔΗΓΟΣ ΧΡΗΣΗΣ

## Επισκόπηση (Overview)

Το **Waste Fee Offset** είναι η επαναστατική υπηρεσία του PowerSave που επιτρέπει στους πολίτες να **μηδενίσουν τα Δημοτικά Τέλη Σκυβάλων** χρησιμοποιώντας την ενέργεια που γλιτώνουν.

**Πώς να μηδενίσετε τα Τέλη Σκυβάλων σας, κλείνοντας απλά τον διακόπτη.**

---

## 🎯 Τι είναι το Waste Fee Offset;

Κάθε φορά που μειώνετε την κατανάλωση ρεύματος σε ώρες αιχμής:
- Το σύστημα **μετατρέπει τις κιλοβατώρες (kWh) σε Ευρώ (€)**
- Τα χρήματα **καταθέτονται αυτόματα** στο ψηφιακό σας πορτοφόλι
- Πληρώνετε τα **Δημοτικά Τέλη** χωρίς να βγάλετε ευρώ από την τσέπη σας!

---

## 🚀 Βήματα Ενεργοποίησης

### Βήμα 1: Σύνδεση Υποστατικού
1. Ανοίξτε την εφαρμογή: `http://localhost:5000/waste_fee_offset.html`
2. Συμπληρώστε τα στοιχεία σας:
   - Ονοματεπώνυμο
   - Αριθμός Υποστατικού (property number)
   - Δήμος (Λευκωσία, Λεμεσός, κλπ.)
   - Διεύθυνση
   - Ετήσιο Τέλος Σκυβάλων (π.χ. €185.00)
3. Πατήστε **"Σύνδεση Λογαριασμού"**

### Βήμα 2: Ορισμός Στόχου
Το σύστημα αναγνωρίζει αυτόματα το ετήσιο τέλος σας και δημιουργεί:
- **Waste Wallet**: Το ψηφιακό σας πορτοφόλι
- **Progress Bar**: Μπάρα προόδου προς τον στόχο
- **Στόχος Σκυβάλων**: 0% Καλυμμένο → 100% Καλυμμένο

## ⚡ Η Διαδικασία Εξοικονόμησης

### 1. Η Ειδοποίηση (The Alert) 🔔
Λαμβάνετε ειδοποίηση συνήθως **17:00 - 20:00**:
> "PowerSave Alert: Η ζήτηση ρεύματος είναι υψηλή. Ξεκινάμε Session Εξοικονόμησης για τις επόμενες 2 ώρες. Συμμετέχεις;"

### 2. Η Δράση (The Action) ⚡
Πατάτε "Συμμετοχή" και κάνετε απλές κινήσεις:
- ✅ Αναβάλετε τη χρήση πλυντηρίου/στεγνωτηρίου
- ✅ Κλείστε τον ηλεκτρικό θερμοσίφωνα
- ✅ Ρυθμίστε το κλιματιστικό στους 26°C (αντί για 22°C)
- ✅ Αποφύγετε τον φούρνο

### 3. Το Αποτέλεσμα (The Reward) 💰
Μόλις τελειώσει το Session:
> "Μπράβο! Κατανάλωσες 2 kWh λιγότερο από το συνηθισμένο σου. Κέρδισες €0.68 στο Waste Wallet σου."

### Πώς γίνεται η εξόφληση;
**Δεν χρειάζεται να κάνετε τίποτα!**

1. Κάθε μήνα το σύστημα αθροίζει τα κέρδη σας (π.χ. €15.00)

3. Λαμβάνετε απόδειξη: *"Πληρώθηκε έναντι Τελών Σκυβάλων: €15.00"*

## ❓ Συχνές Ερωτήσεις (FAQ)

### Τι γίνεται αν μαζέψω περισσότερα χρήματα από τα τέλη μου;
🎉 **Συγχαρητήρια!** Το πλεόνασμα δεν χάνεται:
- **Επιλογή Α**: Μεταφέρεται για τα τέλη του επόμενου έτους
- **Επιλογή Β**: Το δωρίζετε στο "Κοινωνικό Ταμείο" για ευάλωτες οικογένειες

### Επηρεάζεται ο λογαριασμός της ΑΗΚ;
❌ **Όχι!** Πληρώνετε τον λογαριασμό ρεύματος κανονικά στον Πάροχό σας.
- Το PowerSave είναι ένας **επιπλέον κουμπαράς**
- Γεμίζει από την **απουσία κατανάλωσης**

### Πώς ξέρει η εφαρμογή ότι έκανα εξοικονόμηση;
🧠 **Έξυπνος Αλγόριθμος**:
- Συγκρίνει τη σημερινή σας κατανάλωση με τον **Μέσο Όρο** των προηγούμενων 10 ημερών (Baseline)
- Αν καταναλώσατε **λιγότερο** από τον μέσο όρο → η διαφορά είναι το κέρδος σας!

**Baseline Formula:**
```
Baseline = Μέσος Όρος (τελευταίες 10 ημέρες) × 1.3 (για ώρες αιχμής)
Savings = Baseline - Actual Consumption
Earnings (€) = Savings (kWh) × €0.34/kWh
```

## 🔧 Τεχνικά Χαρακτηριστικά

### Backend Engine
**File:** `waste_fee_offset.py`

**Classes:**
- `WasteFeeAccount`: Δημοτικός λογαριασμός
- `SavingSession`: Καταγραφή session
- `WasteWallet`: Ψηφιακό πορτοφόλι
- `WasteFeeOffsetEngine`: Κύριος μηχανισμός υπολογισμών

**Key Features:**
- ✅ Baseline calculation (10-day rolling average)
- ✅ kWh → € conversion (€0.34/kWh)
- ✅ Double points multiplier (x2)
- ✅ Automatic monthly payments
- ✅ Surplus management

### API Endpoints

| Endpoint | Method | Περιγραφή |
|----------|--------|-----------|
| `/api/waste-fee/account/create` | POST | Σύνδεση υποστατικού |
| `/api/waste-fee/wallet/<user_id>` | GET | Κατάσταση πορτοφολιού |
| `/api/waste-fee/session/complete` | POST | Ολοκλήρωση session |
| `/api/waste-fee/payment/process` | POST | Μηνιαία πληρωμή |
| `/api/waste-fee/tips` | GET | Tips εξοικονόμησης |
| `/api/waste-fee/simulate` | GET | Υπολογισμός σεναρίου |

### Frontend UI
**File:** `waste_fee_offset.html`

**Sections:**
1. 🏠 Σύνδεση Υποστατικού (Setup Form)
2. 👛 Waste Wallet Display
3. ⚡ Session Simulator
4. 💡 Energy-Saving Tips
5. 📊 Scenario Calculator

## 🎮 Χρήση της Εφαρμογής

### Τρόπος 1: Standalone Interface
```bash
# Start server
python server.py

# Open browser
http://localhost:5000/waste_fee_offset.html
```

### Τρόπος 2: API Integration
```javascript
// Create account
const account = await fetch('/api/waste-fee/account/create', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        user_id: 'user_123',
        property_number: '12345678',
        municipality: 'nicosia',
        annual_fee: 185.00,
        owner_name: 'Γιώργος Παπαδόπουλος',
        address: 'Λεωφ. Μακαρίου 123'
    })
});

// Complete session
const session = await fetch('/api/waste-fee/session/complete', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        user_id: 'user_123',
        actual_kwh: 1.8,
        historical_consumption: [2.5, 2.3, 2.6, 2.4, 2.5],
        is_double_points: false
    })
});

// Get wallet status
const wallet = await fetch('/api/waste-fee/wallet/user_123');
const data = await wallet.json();
console.log(`Balance: €${data.balance_eur}`);
```

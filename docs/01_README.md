# 🔋 PowerSave Cyprus

> **"Μη παραπονιέσαι για τα τέλη. Μείωσέ τα μόνος σου γυρίζοντας το διακόπτη."**

## Επισκόπηση

Το **PowerSave** είναι το εθνικό πρόγραμμα εξοικονόμησης ενέργειας της Κύπρου που συνδυάζει:
- 🎮 **Gamification** - Παιχνιδοποίηση της εξοικονόμησης
- 👥 **Κοινωνική Κινητοποίηση** - Community challenges και solidarity
- 🔌 **Έξυπνη Τεχνολογία** - Smart meters και AI
- 🏫 **Σχολική Εκπαίδευση** - Reverse socialization μέσω παιδιών

## Το Πρόβλημα που Λύνει

### Πρόβλημα 1: Κύκλος Πόνου των Δημάρχων
```
Υψηλά τέλη → Παράπονα πολιτών → Χαμηλή είσπραξη → 
Δημοτικό έλλειμμα → Αύξηση τελών → Επανάληψη...
```

### Πρόβλημα 2: Εθνικές Δεσμεύσεις ΕΣΕΚ 2021-2030
- **Μέτρο 18**: 15.1% εξοικονόμησης από ευάλωτα νοικοκυριά
- **Μέτρο 4**: Ενεργοποίηση πολιτών (όχι μόνο φυλλάδια)

## Η Καινοτομία: Waste Fee Offset

Το PowerSave μετατρέπει την **εξοικονόμηση ενέργειας** σε **άμεση πληρωμή** για τα δημοτικά τέλη σκυβάλων.

| Βήμα | Περιγραφή |
|------|-----------|
| 1️⃣ | Επιλέγεις Saving Session (π.χ. 17:00-20:00) |
| 2️⃣ | Το σύστημα υπολογίζει το baseline σου |
| 3️⃣ | Εξοικονομείς ενέργεια κάτω από το baseline |
| 4️⃣ | Τα € πιστώνονται αυτόματα στο Waste Wallet |

## Βασικά Νούμερα

| Μετρική | Τιμή | Πηγή |
|---------|------|------|
| Τιμή kWh | €0.34 | ΑΗΚ |
| CO₂/kWh | 0.65 kg | Grid Emission Factor |
| Ετήσια τέλη σκυβάλων | €180-230 | Δήμοι |
| Απαιτούμενη εξοικονόμηση | ~1.85 kWh/ημέρα | Υπολογισμός |

## Χρονοδιάγραμμα

| Φάση | Περίοδος | Στόχος |
|------|----------|--------|
| Phase 1 | Q1 2025 | Pilot σε έναν Δήμο |
| Phase 2 | 2026 | Εθνικό rollout + EU Presidency showcase |
| Phase 3 | 2027 | Πλήρης ενσωμάτωση με Smart Meters |

# PowerSave Cyprus 🇨🇾⚡

## Το Εθνικό Πρόγραμμα Εξοικονόμησης Ενέργειας

Το **PowerSave** είναι μια καινοτόμος πλατφόρμα που συνδυάζει:

- 🎮 **Gamification** - Παιχνιδοποίηση της εξοικονόμησης ενέργειας
- 👥 **Κοινωνική Κινητοποίηση** - Διαγωνισμοί γειτονιών και σχολείων
- 📱 **Έξυπνη Τεχνολογία** - Real-time παρακολούθηση κατανάλωσης

## Στόχος

Μείωση της ενεργειακής κατανάλωσης κατά **15%** έως το 2026, ξεκινώντας από τη Λευκωσία.

## Εργαλεία

- **Waste Fee Offset** - Μηδενισμός τελών σκυβάλων μέσω εξοικονόμησης
- **Solar Sync** - Βελτιστοποίηση χρήσης φωτοβολταϊκών
- **Community Groups** - Διαγωνισμοί κοινοτήτων

# PowerSave Cyprus - Financial Model

## Επισκόπηση Οικονομικού Μοντέλου

Το οικονομικό μοντέλο του PowerSave βασίζεται στη μετατροπή της εξοικονόμησης ενέργειας σε νομισματική αξία που μπορεί να χρησιμοποιηθεί για πληρωμή δημοτικών τελών.

---

## Βασικές Τιμές Αναφοράς

| Παράμετρος | Τιμή | Πηγή |
|------------|------|------|
| **Τιμή μετατροπής kWh** | €0.30/kWh | PowerSave Platform |
| **Αποφυγή CO₂** | 0.65 kg/kWh | ΑΗΚ Grid Emission Factor |
| **Τιμή ρεύματος ΑΗΚ** | ~€0.34/kWh | ΑΗΚ Τιμολόγια 2024 |
| **Ετήσια τέλη σκυβάλων** | €180 - €230 | Δήμοι Κύπρου |

---

## Υπολογισμός Εξοικονόμησης

### Τύπος Υπολογισμού

```
saved_eur = saved_kwh × €0.30
saved_co2_kg = saved_kwh × 0.65
```

### Παράδειγμα Saving Session

| Μέτρηση | Τιμή |
|---------|------|
| Baseline kWh | 3.5 kWh |
| Actual kWh | 2.0 kWh |
| **Saved kWh** | **1.5 kWh** |
| **Saved EUR** | **€0.45** |
| **Saved CO₂** | **0.975 kg** |

---

## Στόχος: Μηδενισμός Τελών Σκυβάλων

### Απαιτούμενη Εξοικονόμηση

Για μηδενισμό ετήσιων τελών **€200**:

```
€200 ÷ €0.30/kWh = 667 kWh/έτος
667 kWh ÷ 365 ημέρες = ~1.83 kWh/ημέρα
```

### Πρακτικά Παραδείγματα

| Ενέργεια | Εξοικονόμηση/ημέρα |
|----------|-------------------|
| Κλείσιμο A/C 1 ώρα (12000 BTU) | 1.15 kWh |
| Πλύσιμο στους 30°C αντί 60°C | 0.8 kWh |
| LED αντί λαμπτήρων πυράκτωσης | 0.3 kWh |
| Standby off (5 συσκευές) | 0.2 kWh |
| **Σύνολο** | **2.45 kWh** ✅ |

## ROI για Συμμετέχοντες

### Προβλεπόμενη Μείωση Κόστους

| Μέτρηση | Πριν | Μετά | Μείωση |
|---------|------|------|--------|
| Κόστος kWh | €0.32 | €0.25 | **-23.19%** |
| Ετήσιο κόστος (5000 kWh) | €1,600 | €1,250 | **-€350** |

### Πρόσθετα Οφέλη

- **Waste Wallet Credits**: Έως €230/έτος
- **Green Points**: Rewards και gamification
- **CO₂ Certificates**: Πιστοποιητικά για ESG reporting

## Δημοτικά Οικονομικά

### Οφέλη για Δήμους

| Τομέας | Όφελος |
|--------|--------|
| **Εγγυημένα έσοδα** | On-Bill Clearinghouse εξασφαλίζει πληρωμές |
| **Μειωμένα παράπονα** | Πολίτες "πληρώνουν" με εξοικονόμηση |
| **SECAP Compliance** | Αυτόματη καταγραφή CO₂ μείωσης |
| **EU Funding** | Πρόσβαση σε Green Deal χρηματοδότηση |

### Cash Flow Model

```
[Καταναλωτής] → Εξοικονόμηση → [PowerSave Platform]
                                      ↓
                              Υπολογισμός Credits
                                      ↓
[ΑΗΚ Bill] ← On-Bill Clearing ← [Waste Wallet]
                                      ↓
                              [Δήμος] ← Earmarked Payment
```

### Projections (2027)

- **Demand Response Capacity**: 4,000 MW
- **Peak Shaving Value**: €50-100/MW/hour
